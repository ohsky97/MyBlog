# MyBlog

이 프로젝트는 개인 공부를 위해 로컬서버로 만든 것을 aws ec2인스턴스 서버에 배포하여
실제로 이용할 때 문제가 없는지 실시간 테스트용으로 만든 사이트입니다.

# 사용 기술

  * back-end
    - java10
    - spring-boot
    - spring-security
    - gradle
    - mybatis
    - jpa
    - aws(EC2, RDS, S3, Route 53)

  * front-end
    - html5
    - css3
    - javascript
    - jquery
    - bootstrap
    
  * Database
    - MySQL
